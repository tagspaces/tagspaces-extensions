# TagSpaces Overview

An open source file manager with tagging

---

## Features

- File tagging with colored tags
- Multiple file format viewers
- Cross-platform support
- Privacy-first approach

---

## Extensions

The extension system provides:

1. Image viewing
2. Document rendering
3. Code editing
4. Media playback

---

## Getting Started

```bash
git clone https://github.com/tagspaces/tagspaces.git
cd tagspaces
npm install
npm start
```
